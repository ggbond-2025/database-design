# 前端运行说明

1. 执行 `npm install`。
2. 执行 `npm run dev`。
3. 打开 Vite 输出的本地地址。

## 验证命令

```powershell
npm run build
```

开发环境默认通过 Vite 代理把 `/api` 转发到 `http://localhost:8080`。

## 页面验证

- `src/main.ts` 已注册 Element Plus 并使用中文语言包，登录页表单组件应渲染为可输入、可点击控件；学生管理编辑入学时间时，日期选择器文案应显示为中文。
- 联调登录前需要先启动后端服务，否则登录请求会由 Vite 代理返回连接失败或 502。

## 协同流程验证

- 管理员端：班级、学生、开课安排、选课记录、成绩和用户账号表单中的关联对象应使用下拉选择；统计查询中的学生和开课安排也应使用下拉选择。
- 管理员端：课程管理维护课程类型、开设年级和开设学期；选课记录管理提供全局选课开关。
- 教师端：从“我的任课”点击“名单”“录成绩”“排名”应自动进入对应页面并带入开课安排。
- 学生端：学生中心展示当前年级学期；可选课程只展示当前年级学期的选修课；我的成绩按年级学期分组；我的学分显示毕业所需和剩余学分；我的班级展示本班学生名单。
